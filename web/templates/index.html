👋 Добро пожаловать в Медицинский бот

📋 О проекте

Умный помощник для пациентов и врачей. Автоматизирует ведение медицинской карты, напоминания о лекарствах и проверку безопасности лечения.

👤 Для пациентов

  • ✓ Напоминания о лекарствах
  • ✓ Дневник симптомов
  • ✓ История назначений

👨‍⚕️ Для врачей

  • ✓ Управление назначениями
  • ✓ Контроль приверженности
  • ✓ Шаблоны рецептов

🏥 Для клиник

  • ✓ Управление пациентами
  • ✓ Отчеты и статистика
  • ✓ Интеграция с МИС

🚀 API Endpoints

Метод Путь Описание
POST /api/register Регистрация
POST /api/login Вход в систему
GET /api/profile Профиль (требуется токен)
GET /health Проверка здоровья

🔧 Тестирование API

Попробуйте зарегистрироваться:

curl -X POST http://localhost:8080/api/register \
  -H "Content-Type: application/json" \
  -d '{
    "email": "test@example.com",
    "password": "password123",
    "first_name": "Иван",
    "last_name": "Петров",
    "role": "patient"
  }'
